# Лабораторная работа 3. ИТМО y29 Разработка ПО

**Этот репозиторий это копия приватного реозитория первой лабораторной работы первого курса ИТМО, программа - разработка ПО**

## Суть работы

Написать парсер аргументов командой строки

## Работа с библиотекой

В ~~класс~~ структуру `ArgumentParser` через соответствующие функции должны быть добавлены _позиционные_ и _именованные_ аргументы и флаги, последующее взаимодействие будет осуществляться через функции `nargparse::GetRepeated` и проч.

## Структура проекта

- `Container` - ~~класс~~ структура-аналог вектора (с урезанным функциналом и совершенно не STL-совместимого), взаимодействие через функции ниже
- `ContRealloc` - реаллоцирует память для `Container`
- `ContPushBack` - добавляет элемент в конец `Container`
- `ArgParser` - основная структура парсера с нужными полями
- `ArgsN` - енум специфицирующий количество аргументов определенного типа/флага
- `Flag` - структура флага
- `ValueType` - енум для спецификации типа аргумента
- `Validator` - валидатор для аргумента произвольного типа (поэтому и `union`)
- `PosArg` - структура позиционного аргумента
- `NamedArgs` - структура именованного аргумента
- `AddFlag` - добавляет флаг в парсер
- `AddArgument` - добавляет аргумент (в зависимости от аргументов позиционный или именованный) в ArgParser
- `Parse` - вызывает все нужные функции для парсинга набора аргументов
- `GetRepeatedCount` - если некоторый аргумент может быть получен несколько раз показывает сколько аргументов получено
- `GetRepeated` - копирует значение аргумента с определенным индексом в `target`
- `AddHelp` и `PrintHelp` - работа с блоком помощи
- `FreeParser` - освобождает всю память выделенную под парсер
